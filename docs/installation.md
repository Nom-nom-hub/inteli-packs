# Installation

## Prerequisites

- Node.js >=16.0.0 or higher
- npm or yarn

## Installation

### Using npm

```bash
npm install inteli-packs
```

### Using yarn

```bash
yarn add inteli-packs
```

### Global installation

```bash
npm install -g inteli-packs
```

## Configuration

Create a `.env` file in your project root:

```env
GEMINI_API_KEY=your_api_key_here
```
