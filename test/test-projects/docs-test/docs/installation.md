# Installation

## Prerequisites

- Node.js 16.0.0 or higher
- npm or yarn

## Installation

### Using npm

```bash
npm install docs-test
```

### Using yarn

```bash
yarn add docs-test
```

### Global installation

```bash
npm install -g docs-test
```

## Configuration

Create a `.env` file in your project root:

```env
GEMINI_API_KEY=your_api_key_here
```
